# 21 JavaScript Projects

Learning JavaScript by building 21 projects in freeCodeCamp

## Concepts so far
1. Functions
2. Arrays:
    - .shift() 
    - .unshift()
    - forEach()
    - .splice()
    - .sort() 
3. Arrow functions
4. .map(): takes a callback function as its first argument. This callback function takes a few arguments, but the first one is the current element being processed
5. .filter(): takes a callback function. The callback function takes the current element as its first argument.
    - allow you to filter elements out of an array, creating a new array in the process.
6. localStorage 
7. Recursion
8. Events (click, change, keydown)
9. .parseInt()
10. regex
11. Sorting Algorithms
    - Bubble Sort
    - Selection Sort
    - Insertion Sort
12. .reduce(): takes an array and applies a callback function to condense the array into a single value.
13. .split(): takes a string and splits it into an array of strings
